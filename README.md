
# Alien Invasion Game

A simple arcade-style game built with Python and Pygame.

## Installation

1. Install Python from [python.org](https://www.python.org/).
2. Install Pygame:
   ```bash
   pip install pygame
   ```

## Run the Game

Run the main script:

```bash
python alien_invasion.py
```

## Controls

- **Move Left/Right**: Arrow keys
- **Quit**: Close window
